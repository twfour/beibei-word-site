# 贝贝外刊汇总项目需求

## 1. 项目目标

把“贝贝外刊”PDF 持续整理为便于检索、阅读、复习和讨论的结构化资料库。项目既要保留每篇文章的独立笔记，也要形成跨日期、跨主题的总览，方便快速了解近期外刊内容及积累英语表达。

## 2. 原始资料

- 当前资料来源：`/Users/apple/Downloads/贝贝外刊`
- 以原始 PDF 为事实依据。
- 新增 PDF 后，应能够增量更新，不重复处理已有内容。
- OCR 或自动提取结果可能遗漏图片、手写内容和复杂版式；重要事实、专有名词及语言点需回看 PDF 核验。

## 3. 单篇文章汇总要求

每篇文章至少整理以下内容：

1. **基本信息**：日期、文章标题、来源文件、主题标签。
2. **新闻切入点（News Hook）**：发生了什么、涉及谁、为什么此时值得关注。
3. **专有名词（Proper Nouns）**：人物、公司、机构、政策、地点等，并给出必要说明。
4. **核心内容摘要**：用简洁中文概括文章主要信息和观点。
5. **因果链（Causal Chain）**：用“因为—所以—但是”等关系梳理事件发展和论证逻辑。
6. **语言笔记（Language Notes）**：高价值词汇、短语、搭配、标题表达和修辞用法；包括英文、中文释义及原文语境。
7. **讨论题（Discussion Prompts）**：提供适合口语、写作或课堂讨论的问题。
8. **核验说明**：标记 OCR 不确定、信息缺失或需要回看原文的位置。

## 4. 跨文章汇总要求

项目应提供以下总览：

- 按日期排列的文章索引。
- 按主题聚合的文章索引。
- 可反查文章的词汇与表达索引。
- 人物、机构、公司及政策等专有名词索引。
- 多篇文章共同涉及的热点、观点差异及事件发展脉络。
- 可用于阶段复习的重点词汇、核心观点和讨论题清单。

## 5. 推荐目录结构

```text
贝贝外刊/
├── README.md                 # 项目说明和使用入口
├── REQUIREMENTS.md           # 本需求文档
├── articles/                 # 单篇文章结构化笔记
├── indexes/                  # 日期、主题、词汇、专有名词索引
├── summaries/                # 周度、月度或专题汇总
└── scripts/                  # 扫描、提取、更新与校验工具
```

## 6. 更新流程

1. 扫描资料目录，识别新增或发生变化的 PDF。
2. 提取文章标题、日期、正文及语言点。
3. 按统一模板生成或更新单篇笔记。
4. 更新日期、主题、词汇和专有名词索引。
5. 对疑似 OCR 错误和关键信息进行原文核验。
6. 输出本次新增、更新、失败及待人工确认的处理报告。

## 7. 输出规范

- 主要成果使用 Markdown，便于阅读、搜索和版本管理。
- 文件名应稳定、可排序，建议采用 `YYYYMMDD-文章短标题.md`。
- 英文原文摘录应适量，优先记录有学习价值的上下文。
- 同一词汇或专有名词采用统一写法，避免产生重复条目。
- 汇总内容应区分原文事实、文章观点与整理者的推断。

## 8. 验收标准

- 每个已处理 PDF 都能对应到一篇文章笔记，并能追溯到源文件。
- 每篇笔记完整包含“新闻切入点—专有名词—因果链—语言笔记—讨论题”。
- 日期、主题、词汇和专有名词索引可以正确跳转或定位到对应文章。
- 重复执行更新不会无故生成重复文章或重复索引项。
- OCR 不确定内容有清晰标记，关键内容已依据 PDF 核验。
- 新增 PDF 后可以只处理增量内容，并产出清晰的更新结果。
